    Tools
    Use the tools to help you understand and manage Peregrine CMS
    • composum
      Direct access to the content repository and package manager of Apache Sling.
    • peregrine api
      Explore the headless API used by Peregrine CMS for all client/server communication.
    • javadocs
      See the java documentation for all server side code of Peregrine CMS

    Peregrine CMS - a head optional content management system based on Apache Sling and Vue.JS.

    For documentation about Peregrine CMS please visit our website.

    Peregrine CMS is an open source project sponsored by headwire.com, Inc. Commercial support is available upon request. We encourage contributions to the project. Please let us know what you think about this product.
